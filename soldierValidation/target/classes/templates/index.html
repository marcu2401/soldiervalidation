Soldier REST API

Web services
http method url path service
GET http://localhost:8080/soldiers all soldiers
GET http://localhost:8080/soldiers/{id} selected soldier
POST http://localhost:8080/soldiers/ recruit new soldier
PATH http://localhost:8080/soldiers/{id}/{shotpeople} update soldier: numbers of shots
PUT http://localhost:8080/soldiers/{id} change properties of soldier
DELETE http://localhost:8080/soldiers/{id} dismantle soldier
